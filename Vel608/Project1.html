The process

Trying to insert myself in to a big wall of code and trying to reverse engineer it seemed overwhelming so I just started from scratch. I ended up liking the cleaner minimalist look a lot more too. This ended up being a mistake as it consumed a lot of time when there wasn't a lot of it, but I also learned a lot about HTML and CSS. First I relied heavily on this video to learn HTML and CSS:
